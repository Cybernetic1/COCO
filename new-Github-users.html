如何注册 GitHub

How to register GitHub

  • Visit https://www.github.com
  • 注册户口（请自行按指示完成这步）
  • 制造个人密码：
    • 在右上角 点击自己的 icon，再点击菜单里的 "settings"
    • 看左边菜单，最底下有 "Developer settings"
    • 进去之后，左边菜单，选 "Personal access tokens"
    • 在新页面里，按 "Generate new token" 按钮
    • Expiration 选择「永不过期」
    • 权限方面选择 "repo"
    • 按最底下 "Generate token" 按钮
    • 用电脑记下这个密码，供以后 GitHub 读写用
  • Visit https://www.github.com
  • Register for an account (please follow their web site's instructions to complete this step)
  • Create a personal password:
    • Upper right corner, click on your icon, click "settings" from the menu
    • Look at menu on the left, the bottom has "Developer settings"
    • Entering, on the left menu, choose "Personal access tokens"
    • In the new page, press "Generate new token" button
    • Choose "Never expire"
    • For access rights choose "repo"
    • Press "Generate token" button at the bottom
    • Record this password in your computer, which is used for reading and writing to GitHub
