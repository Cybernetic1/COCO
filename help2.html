添加 某项工作的作者

先点击一个节点，然后在左边视窗里，Authors 之下可以添加。

投票 作者贡献的百分比 %

先在一件工作里添加若干个 作者，然后点击工作节点，会看到投票界面。

Adding an author to a task

Click on a node first, then on the lower left panel, add authors.

Voting on authors' contribution % percentages

Add some authors to a node first, then clicking on the node will show the voting panel.
